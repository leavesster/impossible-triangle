## 이것은 무엇인가요?

"불가능한 삼각형" 개념을 시각화하는 도구입니다. 세 가지 바람직한 속성을 동시에 달성할 수 없는 상황을 표현합니다. 대표적인 예: 블록체인의 탈중앙화, 보안, 확장성 트릴레마.

## 기능

- **9개 언어 지원** - 브라우저 언어 자동 감지, 수동 전환 가능
- **인터랙티브** - 개념이나 교차점을 클릭하여 편집
- **내보내기** - PNG 이미지로 다운로드
- **의존성 없음** - 단일 HTML 파일, 오프라인 작동

## 사용 방법

1. 모든 최신 브라우저에서 `impossible-triangle.html` 열기
2. 세 가지 핵심 개념 입력
3. "삼각형 생성" 클릭
4. 라벨을 클릭하여 교차 설명 추가
5. 필요시 이미지로 내보내기
