## What is this?

A visual tool to illustrate the "impossible triangle" concept - when three desirable properties cannot be achieved simultaneously. Classic example: blockchain's trilemma of decentralization, security, and scalability.

## Features

- **9 Languages** - Auto-detects browser language, supports manual switching
- **Interactive** - Click on concepts or intersections to edit
- **Export** - Download as PNG image
- **No Dependencies** - Single HTML file, works offline

## Usage

1. Open `impossible-triangle.html` in any modern browser
2. Enter your three core concepts
3. Click "Generate Triangle"
4. Click on labels to add intersection descriptions
5. Export as image if needed
