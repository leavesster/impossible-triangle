## 这是什么？

一个可视化工具，用于展示"不可能三角"概念 - 当三个理想属性无法同时实现时。经典案例：区块链的去中心化、安全性和可扩展性三难困境。

## 功能特性

- **9 种语言** - 自动检测浏览器语言，支持手动切换
- **交互式** - 点击概念或交集区域即可编辑
- **导出功能** - 下载为 PNG 图片
- **零依赖** - 单个 HTML 文件，可离线使用

## 使用方法

1. 在任意现代浏览器中打开 `impossible-triangle.html`
2. 输入你的三个核心概念
3. 点击"生成三角图"
4. 点击标签添加交集描述
5. 需要时导出为图片
