## これは何ですか？

「不可能三角形」のコンセプトを視覚化するツールです。三つの望ましい特性を同時に達成できない状況を表現します。代表例：ブロックチェーンの分散化、セキュリティ、スケーラビリティのトリレンマ。

## 機能

- **9言語対応** - ブラウザ言語を自動検出、手動切り替えも可能
- **インタラクティブ** - コンセプトや交差部分をクリックして編集
- **エクスポート** - PNG画像としてダウンロード
- **依存関係なし** - 単一のHTMLファイル、オフラインで動作

## 使い方

1. 任意のモダンブラウザで `impossible-triangle.html` を開く
2. 三つのコア概念を入力
3. 「三角形を生成」をクリック
4. ラベルをクリックして交差説明を追加
5. 必要に応じて画像としてエクスポート
